# 🏛️ Data Engineering Pipeline - Reservasi Gedung Pernikahan

## 📌 Overview
Project ini membangun pipeline data sederhana untuk mengolah data reservasi gedung pernikahan menjadi data yang bersih, aman, dan siap dianalisis.

Pipeline mencakup proses:
- Data Cleansing
- Data Quality Check
- Data Masking (Data Governance)
- Data Storage (MySQL & Excel)
